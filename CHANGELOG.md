# Changelog

All notable changes to this project will be documented in this file.

## [v1.10] - 2026-04-28

### Added
- Drag-and-drop batch conversion GUI for MarkItDown.
- OCR fallback for scanned/image-based PDFs.
- Traditional Chinese OCR cleanup rules.
- Financial table mode for better numeric row formatting.
- Dropped-file name preview in GUI.
- Timeout control and fast mode (`convert_local`).
- Output folder remember option.
- MIT `LICENSE` and `requirements.txt`.
- Public `README.md` with install, usage, and build guide.

### Release
- Windows executable published:
  - `BatchMarkItDownGUI_v110.exe`
  - Release tag: `v1.10`
